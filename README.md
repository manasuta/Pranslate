# Pranslate

**Pranslate** は、英語のプログラミング学習教材（ドキュメント・チュートリアル・エラーメッセージなど）を読む日本語話者向けの、Chrome 拡張機能（Manifest V3）です。

ページ上でテキストを選択するだけで、小さなポップアップから「日本語訳」「コード説明」「用語説明」の3つのモードを選び、選択箇所の近くに表示されるカードに Gemini API による回答をストリーミング表示します。

## 主な機能

- **3つのモード**
  - 日本語訳: 選択した英文を自然な日本語に翻訳
  - コード説明: 選択したコードが何をしているかを簡潔に説明
  - 用語説明: 選択した専門用語・概念をやさしく解説
- **選択ポップアップ**: テキストを選択すると近くに3つのモードボタンが表示される（コードらしい選択は「コード説明」を自動で強調表示）
- **ストリーミングカード**: 回答は Markdown としてリアルタイムに描画され、ドラッグで移動可能。別モードへの切り替え、コピー、再試行にも対応
- **ページ単位のオン/オフ**: 拡張機能アイコンやショートカットで、必要なときだけ有効化できる
- **キャッシュ**: 同じモード・同じテキストの再質問はキャッシュから即座に返し、無駄な API 呼び出しを削減
- **1日あたりの呼び出し上限**: オプションで日次の呼び出し回数に上限を設定し、意図しない課金を防止

## セットアップ

### 1. Gemini API キーを取得する

1. [Google AI Studio](https://aistudio.google.com/app/apikey) にアクセスし、Google アカウントでログインします。
2. 「Create API key」から新しい API キーを発行します。
3. 発行された API キーをコピーしておきます。

### 2. 拡張機能を Chrome に読み込む（開発者モード）

1. Chrome で `chrome://extensions` を開きます。
2. 右上の「デベロッパーモード」をオンにします。
3. 「パッケージ化されていない拡張機能を読み込む」をクリックします。
4. このプロジェクトのフォルダ（`manifest.json` があるフォルダ）を選択します。
5. 一覧に "Pranslate" が追加されれば完了です。

### 3. API キーを設定する

1. 拡張機能の詳細画面、または拡張機能アイコンのメニューから「オプション」を開きます。
2. 「Gemini API キー」欄に、手順1で取得した API キーを入力します。
3. 必要であればモデルや既定のオン/オフ、1日あたりの呼び出し上限も設定します。
4. 「保存」をクリックします。

## 使い方

1. 拡張機能を有効にした状態で、英語のページ上の気になる英文・コード・用語を選択します。
2. 選択範囲の近くに表示されるポップアップから、モードを選びます。
   - 日本語訳 / コード説明 / 用語説明
3. 選択範囲付近にカードが表示され、Gemini の回答がストリーミングで表示されます。
4. カードのヘッダーをドラッグして好きな位置に移動したり、フッターのボタンで別モードに切り替えたり、コピー ボタンで Markdown をコピーできます。

## キーボードショートカット

選択したテキストに対して、マウスでポップアップを開かずに直接モードを実行できます。

| モード | Mac | Windows / Linux |
|---|---|---|
| 日本語訳 | `⇧⌘H` | `Ctrl+Shift+H` |
| コード説明 | `⇧⌘J` | `Ctrl+Shift+J` |
| 用語説明 | `⇧⌘K` | `Ctrl+Shift+K` |

ショートカットが他の拡張機能やブラウザ機能と競合する場合は、`chrome://extensions/shortcuts` から自由に変更できます。変更後は、選択ポップアップに表示されるショートカット表記も**実際の割り当てに自動で同期**されます（未割り当てのモードは表記が消えます）。

> **⚠️ 既定ショートカットの衝突について**
> `⇧⌘J`（Mac）と `Ctrl+Shift+J`（Windows/Linux）は、ブラウザ標準の機能（Mac: ダウンロード表示 / Windows・Linux: デベロッパーツール）と重複します。この場合 Chrome はコード説明のショートカットを**自動割り当てしません**（未設定のまま読み込まれます）。コード説明をショートカットで使いたい場合は、`chrome://extensions/shortcuts` を開いて Pranslate の「コード説明」に**衝突しない任意のキー**を割り当ててください（ポップアップのボタンからはいつでも実行できます）。

## プライバシーについて

- この拡張機能のコンテンツスクリプトは、どのページでも選択ポップアップを利用できるように `<all_urls>` に対して動作します。ページの内容を収集・送信することはなく、**あなたが実際にモードを実行したときのみ**、以下の情報が Google の Gemini API に送信されます。
  - 選択したテキスト
  - 選択箇所を含む周辺の段落（文脈として、一定文字数まで）
  - ページタイトルおよび URL
- これら以外のページ内容やブラウジング履歴が外部に送信されることはありません。
- API キーやキャッシュ、利用状況（呼び出し回数）は Chrome の `chrome.storage.local`（ローカルストレージ）にのみ保存され、拡張機能の作者を含む第三者には送信されません。

## モデルについて

既定のモデルは `gemini-2.5-flash-lite` です。最も高速・軽量で、無料枠の1日あたり呼び出し回数（RPD）が最大のため、無料で使い倒すのに向いています。より高い品質が欲しい場合はオプションページの「モデル」欄から `gemini-2.5-flash` などに変更したり、「カスタム...」を選んで任意のモデル ID を指定することもできます。
